## DBeaver 에서 실행 쿼리 이력을 확인하는 방법 메모
위치: `Window -> Show View -> Query Manager (쿼리 관리자)`

Query Manager 기능은 현재 세션 동안 실행한 모든 SQL 쿼리 기록을 보여줌.

그래서 실행한 쿼리를 다시 보고 싶으면 해당 메뉴에서 확인할 수 있다.
