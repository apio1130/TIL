### Junit4와 5 비교

1. `@BeforeClass` -> `@BeforeAll`  
  테스트 클래스 테스트 시작 시 1회 실행
1. `@AfterClass` -> `@AfterAll`  
  테스트 클래스 테스트 종료 시 1회 실행
1. `@Before` -> `@BeforeEach`  
  각각의 테스트 케이스의 시작 전 실행
1. `@After` -> `@AfterEach`  
  각각의 테스트 케이스의 완료 후 실행
1.`@Category` -> `@Tag`
  테스트 필터링을 위한 태그 선언
1.`@Ignore` -> `@Disabled`
  테스트 클래스 또는 메서드를 비활성화
