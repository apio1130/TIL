# MAC 터미널 명령어 컬러 적용

## 설치
아래 명령어를 실행하여 설치
```
brew install zsh-syntax-highlighting
```

## 적용 방법
설치 경로가 다를 수 있기에 brew info에 나오는 정보로 vi를 이용하여 syntax highlighting 적용
```
brew info zsh-syntax-highlighting
```

실행 시 나오는 정보 중 아래 문구의 정보를 따라 `vi ~/.zshrc`를 실행하여 `source ...` 정보를 추가
