# Oracle 인덱스 조회 관련 메모

|테이블|설명|
|---|---|
|ALL_INDEXES|현재 사용자가 액세스 할 수있는 테이블의 색인을 설명합니다.|
|ALL_IND_COLUMNS|현재 사용자가 액세스 할 수있는 모든 테이블의 인덱스 열을 설명합니다.|
