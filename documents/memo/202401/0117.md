# Oracle DB 사용 환경에서 Virtual Thread 사용 가능 조사
- Oracle JDBC 릴리즈 21.1 버전에서 Virtual Thread 를 지원하기 시작
- Oracle 21C 이상 사용하며, OJDBC 11 이상의 드라이버를 사용하면 개발 가능
- Maria DB 는 큰 이슈 안보이는 듯
